# DoraHacks listing — draft

DoraHacks (`dorahacks.io`) is a builder-aggregator with quadratic
funding rounds and bounty boards. Compass fits the
"privacy infrastructure" + "verifiable credentials" categories. The
audience overlap with the 0G ecosystem is high.

## Quadratic funding pitch

DoraHacks runs periodic quadratic-funding rounds. If Compass qualifies
for an upcoming round, the pitch (~80 words) is:

> Compass is the only project we found that takes the migrant-worker
> threat model — abusive employer with subpoena power — as a first-class
> design constraint. The output is a working privacy-preserving
> eligibility check on 0G mainnet that NGOs can drop into their intake
> flow without writing crypto code. Funding extends three concrete
> roadmap items: browser-side 0G Storage upload, on-chain trust-list
> governance, and native-speaker localization review for the five
> kiosk languages we already ship in AI-generated baseline.
