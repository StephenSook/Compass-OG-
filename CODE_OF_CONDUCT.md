# Code of Conduct

Compass exists to protect vulnerable people. We extend that posture to
our project space.

## Standards

Everyone participating in this project is expected to:

- Treat others with respect, especially across differences in
  experience, background, technical level, and geography.
- Assume good intent. If a contribution lands poorly, ask before
  attacking.
- Disagree with ideas, never with people.
- Avoid sharing personal information about identifiable migrant
  workers, NGO clients, or anyone the project is intended to serve —
  including in test fixtures, demo flows, and screenshots. The
  Maria-persona used in the demo is a composite, not a real worker;
  please do not use real names, HKIDs, or employer information when
  contributing tests, examples, or documentation.

## Unacceptable behaviour

- Harassment, discrimination, or personal attacks against any
  contributor.
- Sharing or soliciting real identifying data about migrant workers.
- Using the project for targeted surveillance, discrimination, or any
  use case that materially harms the populations Compass is intended
  to protect.

## Enforcement

If you observe behaviour that violates this code, please contact the
maintainer directly:

- Email: `stephensookra@gmail.com` (use subject `[Compass CoC]`)
- Telegram: `@stephensookra`

The maintainer will review the report within 72 hours, work with all
parties involved, and may take any action deemed appropriate including
a warning, a temporary contribution ban, or a permanent ban.
